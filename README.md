# Small_projects

# Overview
This repository contains two small projects: a Calculator App and a Simon Says Game, built using HTML, CSS, and JavaScript.

# Calculator App
A simple, functional calculator with an appealing design, supporting basic arithmetic operations.

# Features
1. Addition, subtraction, multiplication, and division.
2. Appealing design.
3. Interactive button animations.

# Installation
Clone the repository and open index.html in your browser.

# Technologies Used
1. HTML
2. CSS
3. JavaScript

# Simon Says Game
An interactive memory game where players must replicate a growing sequence of tile presses.

# Features
1. Four colored tiles blinking in sequence.
2. Increasing difficulty with each level.
3. Game over upon incorrect input with restart option.

# Installation
Clone the repository and open index.html to play.

# Technologies Used
1. HTML
2. CSS
3. JavaScript
